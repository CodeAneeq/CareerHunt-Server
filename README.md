# Career Hunt - Job Portal - Backend

This is the **backend (server-side)** for the **Career Hunt Job Portal**, built with **Node.js**, **Express.js**, and **MongoDB**.

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- JWT
- bcrypt
- dotenv
- CORS

---

## 📸 Features

- Job Posting APIs
- Job Application APIs
- User Authentication
- Admin Routes (Optional)
- Secure REST APIs
